## 🚀 Aperçu

BAWON+ est une plateforme immersive mettant en avant la richesse culturelle haïtienne à travers une expérience digitale moderne et élégante.

## 🎯 Vision

BAWON+ ambitionne de devenir une référence mondiale dans la valorisation de l’excellence haïtienne à travers une offre premium combinant :

* Produits artisanaux
* Expériences culturelles
* Lifestyle haut de gamme
